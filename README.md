# jeanjang.com

Welcome to my personal blog, crafted using [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [Contentlayer](https://contentlayer.dev/).

---

## Getting Started

### Installtaion
```bash
yarn install
```

### Development
```bash
yarn dev
```
Then you could see the blog at local:3000
